# 🍦 Ice Cream Rating Data Analysis

## 📌 Project Overview

This project analyzes ice cream ratings and customer preferences using data analytics techniques. The goal is to identify factors that influence ratings, understand customer behavior, and derive actionable insights through exploratory data analysis (EDA) and visualization.

## 🎯 Objectives

- Analyze customer ratings for different ice cream flavors.
- Identify top-rated and lowest-rated products.
- Explore relationships between price, flavor, brand, and ratings.
- Visualize trends and customer preferences.
- Generate business insights for product improvement.

## 📂 Dataset

The dataset contains information such as:

| Column | Description |
|----------|-------------|
| Flavor | Ice cream flavor |
| Brand | Brand name |
| Rating | Customer rating |
| Price | Product price |
| Reviews | Number of reviews |
| Category | Ice cream category |

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## 📊 Exploratory Data Analysis

The analysis includes:

- Data Cleaning
- Missing Value Treatment
- Descriptive Statistics
- Rating Distribution Analysis
- Brand-wise Performance
- Flavor Popularity Analysis
- Correlation Analysis
- Data Visualization

## 📈 Visualizations

- Rating Distribution Histogram
- Top Rated Flavors
- Brand Comparison Charts
- Correlation Heatmap
- Review Count vs Rating Scatter Plot

## 🔍 Key Insights

- Vanilla and Chocolate flavors received the highest average ratings.
- Premium brands generally achieved better customer satisfaction.
- Products with more reviews tend to have more stable ratings.
- Price does not always correlate with higher ratings.
